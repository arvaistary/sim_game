# План разработки (Roadmap)

**Последнее обновление:** 10 апреля 2026
**Версия:** 3.0
**Технологический стек:** Nuxt 4 + Vue 3 + TypeScript + Pinia

---

## Обзор проекта

Проект Game Life - это симулятор жизни с пошаговым геймплеем. Текущая архитектура:

- **4 архитектурных слоя:** Domain (ECS), Application, Infrastructure, Presentation
- **18 ECS систем:** управление всем игровым процессом
- **15 Vue страниц:** весь пользовательский интерфейс
- **~222 действия в 10 категориях:** полная система восстановления
- **TypeScript:** строгая типизация на всех уровнях

## Рекомендации по разработке

### Для добавления новой механики

**Domain Layer:**
- Создайте ECS систему в `src/domain/ecs/systems/`
- Добавьте компоненты ECS в `src/domain/ecs/components/`
- Обновите Game Facade (команды/запросы)
- Добавьте в `src/domain/balance/` балансовые данные

**Application Layer:**
- Добавьте Use Case в `src/application/game/commands.ts`
- Добавьте запрос в `src/application/game/queries.ts`
- Добавьте типы в `src/application/game/types.ts`

**Presentation Layer:**
- Создайте Vue компонент страницы в `src/pages/`
- Создайте composable для логики UI в `src/composables/`
- Добавьте computed свойства в store при необходимости
- Добавьте маппинг в `src/nuxt-pages/game/[section].vue`

**Testing:**
- Unit тесты ECS систем в `test/unit/domain/ecs/`
- Integration тесты для composables
- E2E тесты для критических путей
