# Система навыков (Skills System)

## Обзор

Система навыков позволяет игроку просматривать свои способности и планировать развитие персонажа. Навыки разделены на базовые и профессиональные.

## Структура навыков

### Базовые навыки (Basic Skills) - 10 штук

Фундаментальные навыки, которые формируют основу развития персонажа:

| Ключ | Название | Описание |
|------|----------|----------|
| `timeManagement` | Тайм-менеджмент | Умение планировать и распределять время |
| `communication` | Коммуникация | Способность общаться и договариваться |
| `financialLiteracy` | Финансовая грамотность | Понимание основ управления деньгами |
| `healthyLifestyle` | Здоровый образ жизни | Привычки, поддерживающие здоровье |
| `adaptability` | Адаптивность | Способность быстро адаптироваться к изменениям |
| `discipline` | Дисциплина | Способность следовать плану и правилам |
| `physicalFitness` | Физическая форма | Общая физическая подготовка |
| `emotionalIntelligence` | Эмоциональный интеллект | Понимание собственных и чужих эмоций |
| `organization` | Организованность | Умение поддерживать порядок в делах |
| `basicCreativity` | Базовая креативность | Способность мыслить нестандартно |

### Профессиональные навыки (Professional Skills) - 12 штук

Специализированные навыки для карьерного роста:

| Ключ | Название | Описание |
|------|----------|----------|
| `professionalism` | Профессионализм | Качественное выполнение рабочих задач |
| `leadership` | Лидерство | Способность вести за собой других |
| `negotiations` | Переговоры | Умение договариваться и убеждать |
| `analyticalThinking` | Аналитическое мышление | Способность анализировать информацию |
| `specialization` | Специализация | Глубокие знания в конкретной области |
| `creativity` | Креативность | Профессиональное творчество |
| `stressResistance` | Стрессоустойчивость | Способность работать под давлением |
| `advancedHealth` | Продвинутое здоровье | Специальные знания о здоровье |
| `socialConnections` | Социальные связи | Полезные знакомства и нетворкинг |
| `technicalLiteracy` | Техническая грамотность | Работа с технологиями |
| `cooking` | Кулинария | Умение готовить |
| `marketing` | Маркетинг | Продвижение и продажи |

## Диапазон значений

Все навыки имеют диапазон от **0 до 10** включительно.

- **0** - навык не развит
- **5** - средний уровень развития
- **10** - мастерство

## Начальные значения

При создании персонажа начальные значения навыков зависят от выбранного пути образования:

### "Сразу в жизнь" (нет образования)
```javascript
{
  timeManagement: 1,
  communication: 1,
  financialLiteracy: 1,
}
```

### "Пройти школу" (среднее образование)
```javascript
{
  timeManagement: 1,
  communication: 1,
  financialLiteracy: 1,
  healthyLifestyle: 1,
}
```

### "Пройти школу + Институт" (высшее образование)
```javascript
{
  timeManagement: 1,
  communication: 1,
  financialLiteracy: 1,
  healthyLifestyle: 1,
  professionalism: 1,
}
```

## Использование в карьере

Навыки влияют на доступность работ. Например:

| Работа | Требуемые навыки |
|--------|------------------|
| Офисный сотрудник | Нет требований |
| Координатор проектов | Профессионализм ≥ 3 |
| Бизнес-аналитик | Профессионализм ≥ 4, Среднее образование |
| Тимлид | Профессионализм ≥ 6, Высшее образование |

## UI - Сцена SkillsScene

### Навигация
- Доступ через главную навигацию: **Навыки** (иконка "Н")
- Разделение на две вкладки: **Базовые** и **Профессиональные**

### Элементы карточки навыка
1. **Название** навыка (крупный текст)
2. **Описание** навыка (мелкий текст)
3. **Значение** навыка (X/10 справа вверху)
4. **Прогресс-бар** визуализирует уровень развития

### Файлы

- **`src/scenes/SkillsScene.js`** - основная сцена отображения навыков
- **`src/balance/skills-constants.js`** - константы и определения навыков (подписи, цвета, вкладки UI)
- **`src/ecs/systems/SkillsSystem.js`** - ECS система управления навыками

## Интеграция с ECS

Система навыков полностью интегрирована с ECS:

```javascript
// Получить текущие навыки
const skills = skillsSystem.getSkills();

// Проверить уровень конкретного навыка
const hasRequiredSkill = skillsSystem.hasSkillLevel('professionalism', 3);

// Применить изменения к навыкам
skillsSystem.applySkillChanges({
  professionalism: 1,
  communication: -1,
});
```

## Требования к работам

Система навыков проверяется в `CareerProgressSystem`:

```javascript
// Разблокировка работы
unlocked: professionalism >= job.minProfessionalism && 
          educationRank >= job.minEducationRank
```
