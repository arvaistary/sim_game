# Документация по сценам старта игры

**Дата создания:** 7 апреля 2026  
**Версия игры:** 0.2.0

---

## Обзор

В этот релиз добавлена полноценная система создания персонажа и начального образования. Игрок теперь может:
- Создать персонажа с именем и возрастом
- Выбрать путь образования (без образования, школа, школа+институт)
- Пройти интерактивные мини-игры для получения образования

---

## StartScene

**Файл:** `src/scenes/StartScene.js`  
**Ключ сцены:** `StartScene`

### Назначение

Первая сцена игры, где игрок создаёт своего персонажа и выбирает начальный путь развития.

### Функциональность

#### 1. Ввод имени персонажа

- **UI:** Панель с полем ввода
- **Ввод:** При клике открывается браузерный `prompt()`
- **Валидация:** 
  - Минимум 2 символа
  - Максимум 20 символов
  - Не пустое значение
- **Отображение:** Введённое имя показывается в поле

#### 2. Выбор начального возраста

- **Диапазон:** 18-30 лет
- **Управление:** Кнопки `-` и `+`
- **Отображение:** Крупный текст с текущим возрастом
- **Подсказка:** "Рекомендуется 18-25 лет"

#### 3. Выбор пути образования

Три варианта с разными результатами:

| Путь | Образование | Навыки | Длительность | Сцены |
|------|-------------|---------|---------------|--------|
| Сразу в жизнь | Нет | timeManagement +1, communication +1, financialLiteracy +1 | Мгновенно | MainGameScene |
| Пройти школу | Среднее | Все базовые +1, healthyLifestyle +1 | ~10-12 мин | SchoolIntroScene → MainGameScene |
| Школа + Институт | Высшее | Все базовые +1, professionalism +1 | ~20-22 мин | SchoolIntroScene → InstituteIntroScene → MainGameScene |

#### 4. Кнопка "Начать игру"

- **Валидация:** Перед запуском проверяет:
  - Имя введено и валидно
  - Путь образования выбран
- **Действия при запуске:**
  1. Создаёт новый объект `saveData` на основе `DEFAULT_SAVE`
  2. Применяет выбранные значения (имя, возраст, навыки, образование)
  3. Сохраняет в `localStorage`
  4. Перенаправляет на соответствующую сцену

### Технические детали

- **Адаптивность:** Работает на мобильных и десктопных устройствах
- **Ширина формы:** Максимум 500px, отступы 24px от краёв
- **Отступы между элементами:** 20px
- **Цвета:** Использует константы из `ui-kit.js`

## SchoolIntroScene

**Файл:** `src/scenes/SchoolIntroScene.js`  
**Ключ сцены:** `SchoolIntroScene`

### Назначение

Мини-игра для получения среднего образования. Игрок проходит 4 учебных предмета в формате викторин.

### Структура мини-игры

#### Раунд 1: Математика
- **Длительность:** 60 секунд
- **Вопросов:** 5
- **Тип:** Быстрый выбор правильного ответа
- **Награда:** timeManagement +1
- **Примеры:** Сложение, умножение, деление

#### Раунд 2: Русский язык
- **Длительность:** 90 секунд
- **Вопросов:** 5
- **Тип:** Выбор определения для слова
- **Награда:** communication +1
- **Примеры:** Синонимы, антонимы, части речи

#### Раунд 3: История
- **Длительность:** 90 секунд
- **Вопросов:** 5
- **Тип:** Викторина по истории России
- **Награда:** financialLiteracy +1
- **Примеры:** Даты, события, личности

#### Раунд 4: Биология
- **Длительность:** 90 секунд
- **Вопросов:** 5
- **Тип:** Вопросы по анатомии и биологии
- **Награда:** healthyLifestyle +1
- **Примеры:** Хромосомы, органы, витамины

### Механики

#### Таймер
- Обратный отсчёт для каждого раунда
- Отображается в формате `mm:ss`
- При истечении времени раунд завершается

#### Прогресс
- Визуальный индикатор 4 раундов (кружки)
- Пройденные раунды отмечаются зелёным цветом
- Текущий раунд не отмечен

#### Система очков
- Правильный ответ: +10 очков
- Неправильный ответ: 0 очков
- Проход раунда: ≥50% правильных ответов

#### Награды
- При успешном прохождении раунда: +1 к соответствующему навыку
- Изменения применяются к `saveData.skills`
- Результат сохраняется после каждого раунда

### Завершение

По окончании всех 4 раундов:
- Отображается итоговый счёт
- Устанавливается `education.school = 'completed'`
- Устанавливается `education.educationLevel = 'Среднее'`
- Сохранение в `localStorage`
- Переход в целевую сцену (`MainGameScene` или `InstituteIntroScene`)

## InstituteIntroScene

**Файл:** `src/scenes/InstituteIntroScene.js`  
**Ключ сцены:** `InstituteIntroScene`

### Назначение

Мини-игра для получения высшего образования. Более сложные задачи, чем в школе.

### Структура мини-игры

#### Раунд 1: Бизнес-кейс
- **Длительность:** 120 секунд
- **Вопросов:** 3
- **Тип:** Ситуационные задачи с выбором решения
- **Награда:** professionalism +1
- **Примеры:** Управление командой, работа с клиентами

#### Раунд 2: Управление проектом
- **Длительность:** 180 секунд
- **Вопросов:** 3
- **Тип:** Вопросы о проектном менеджменте
- **Награда:** communication +1
- **Примеры:** Дедлайны, команды, приоритеты

#### Раунд 3: Презентация
- **Длительность:** 150 секунд
- **Вопросов:** 3
- **Тип:** Вопросы о коммуникации и подготовке
- **Награда:** timeManagement +1
- **Примеры:** Структура презентации, аудитория

#### Раунд 4: Финальный проект
- **Длительность:** 200 секунд
- **Вопросов:** 4
- **Тип:** Комплексные бизнес-ситуации
- **Награда:** financialLiteracy +1
- **Примеры:** Стартапы, KPI, инвестирование

### Отличия от школы

| Характеристика | Школа | Институт |
|---------------|--------|-----------|
| Длительность раундов | 60-90 сек | 120-200 сек |
| Количество вопросов | 5 | 3-4 |
| Сложность | Базовая | Продвинутая |
| Очки за правильный ответ | +10 | +15 |
| Размер кнопок ответов | 200×48 | 300×72 |
| Размер текста | 20px | 18px |

## Интеграция с MainGameScene

### Кнопка "Новая игра"

**Расположение:** Верхняя панель (topCard), рядом с кнопкой "Карьера"

**Функциональность:**
- При нажатии открывает модальное окно подтверждения
- Текст: "Вы уверены, что хотите начать новую игру? Весь текущий прогресс будет потерян."
- При подтверждении:
  - Очищает `localStorage` (опционально)
  - Запускает `StartScene`

**Позиционирование:**
- Десктоп: Слева от кнопки карьеры, ниже работы
- Мобильный: По центру под работой

## Потоки пользователя

### Поток 1: Без образования

```
StartScene 
  → Ввести имя 
  → Выбрать возраст 
  → Выбрать "Сразу в жизнь"
  → MainGameScene
```

### Поток 2: Только школа

```
StartScene
  → Ввести имя
  → Выбрать возраст
  → Выбрать "Пройти школу"
  → SchoolIntroScene
    → Раунд 1 (Математика)
    → Раунд 2 (Русский язык)
    → Раунд 3 (История)
    → Раунд 4 (Биология)
  → MainGameScene
```

### Поток 3: Полное образование

```
StartScene
  → Ввести имя
  → Выбрать возраст
  → Выбрать "Школа + Институт"
  → SchoolIntroScene
    → 4 раунда школы
  → InstituteIntroScene
    → Раунд 1 (Бизнес-кейс)
    → Раунд 2 (Управление проектом)
    → Раунд 3 (Презентация)
    → Раунд 4 (Финальный проект)
  → MainGameScene
```

## Известные ограничения

1. **Ввод имени:** Использует браузерный `prompt()`, который нельзя стилизовать
2. **Без выбора работы:** Работа выбирается только в экране карьеры, не при старте
3. **Фиксированные вопросы:** Вопросы в мини-играх не генерируются динамически
4. **Нет отмены:** Нельзя вернуться назад после выбора образования
