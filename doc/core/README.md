# Game Life - Документация для разработчиков

Добро пожаловать в проект Game Life! Это симулятор жизни с пошаговым геймплеем, реализованный на Phaser.js.

## 📚 Быстрый старт

Для новых разработчиков рекомендуем следующий порядок чтения:

1. **🎯 Обзор проекта** - эта страница
2. **📊 Статус реализации** - что готово, что в работе
3. **🧩 Справочник сцен** - точка входа, список `src/scenes/`
4. **🛣️ Roadmap разработки** - план развития
5. **🎮 GDD (Game Design Document)** - полное описание механик игры
6. **📄 Сцены старта игры** - описание StartScene, SchoolIntroScene, InstituteIntroScene
7. **🧠 MemPalace** - локальная память проекта для AI
8. **⚙️ Архитектура ECS** - техническая архитектура

## 📁 Структура документации

### 📖 Основная документация (эта папка)

В этой папке собрана основная документация для входа в проект:

- **README.md** - этот файл, обзор и навигация

## 🚀 Начало работы

### Требования

- Node.js (версия 16+)
- Современный браузер (Chrome, Firefox, Safari, Edge)

### Установка

```bash
# Клонирование репозитория
git clone <repository-url>
cd game_life

# Установка зависимостей (если есть package.json)
npm install
```

### Запуск проекта

```bash
npm install
npm run dev
```

Vite выведет адрес (часто `http://localhost:5173/`). Откройте его в браузере.

## 🎯 Ключевые механики игры

### Core Loop

1. **Создание персонажа** - игрок вводит имя, выбирает возраст и путь образования (StartScene)
   - 3 варианта образования: без образования, школа, школа+институт
   - Мини-игры для получения начальных навыков
2. **Работа** - игрок нажимает "Начать рабочий период"
3. **Восстановление** - игрок тратит деньги на восстановление шкал
4. **Повтор** - цикл повторяется

### Шкалы персонажа

- Голод (Hunger)
- Энергия (Energy)
- Стресс (Stress)
- Настроение (Mood)
- Здоровье (Health)
- Физическая форма (Physical)

### Основные системы

1. **Создание персонажа** - игрок вводит имя, выбирает возраст и путь образования (StartScene)
   - 3 варианта образования: без образования, школа, школа+институт
   - Мини-игры для получения начальных навыков
2. **Работа** - игрок нажимает "Начать рабочий период"
3. **Восстановление** - игрок тратит деньги на восстановление шкал

### Дополнительные системы

- Карьера - progression через уровни работ
- Образование - книги, курсы, вузы
- Дом - жильё и мебель
- Отношения - семья и друзья
- Инвестиции - пассивный доход
- Хобби - побочный заработок

## 🤝 Вклад в проект

### Для разработки новых функций

1. Изучите соответствующий модуль в GDD
2. Проверьте IMPLEMENTATION_STATUS.md для понимания контекста
3. Следуйте ECS архитектуре для новой логики
4. Добавьте тесты при необходимости

### Для исправления багов

1. Воспроизведите проблему
2. Найдите соответствующий код в `src/`
3. Проверьте ECS систем, которые могут влиять на проблему
4. Создайте фикс с тестом
